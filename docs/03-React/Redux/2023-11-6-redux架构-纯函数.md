## 纯函数
- 纯函数指的是指只依赖**自身参数**的函数
  
## Redux规则
1. 所有状态的改变, 只能通过纯函数完成
2. 函数中不允许有**副作用**

## 常见副作用
1. 异步操作
2. 网络请求
3. 文件读写
   
## 深入
### 为什么只能用纯函数
1. **基于Flux架构模式 ->  Flux架构模式优势和思想 -> 和传统MVC架构区别**
   
### flux优势
1. flux希望做到的
   - **Use explicit data instead of derived data**
   - **Separate data from view state**
2. flux希望带来的好处  
   - **Improved data consistency**
   - **Easier to pinpoint root of a bug**
   - **More meaingful unit tests**
